# 🛒 NexCart - Full Stack E-Commerce Website

A modern Full Stack E-Commerce website developed using PHP, MySQL, HTML, CSS and JavaScript. The project includes user authentication, shopping cart, checkout, order management and responsive user interface.

---

## 🚀 Features

- User Registration & Login
- Secure Session Authentication
- Product Listing
- Product Search
- Add to Cart
- Buy Now
- Quantity Management
- Checkout System
- Order Confirmation
- My Orders
- User Profile
- Responsive Design

---

## ⚙ Installation

### 1 Clone Repository

git clone https://github.com/YOUR_USERNAME/NexCart.git

### 2 Move Project

Copy the project into:

xampp/htdocs/

### 3 Start XAMPP

Start

- Apache
- MySQL

### 4 Create Database

Create database:

nexcart

### 5 Import Database

Import

database/nexcart.sql

using phpMyAdmin.

### 6 Run Project

http://localhost/NexCart
